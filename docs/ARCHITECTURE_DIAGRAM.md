## Key Components Summary

### WLEDJsonApiClient
- **Purpose**: Direct JSON API communication
- **Key Methods**: 
  - `set_individual_leds()`
  - `set_led()`
  - `set_led_range()`
- **Features**: Auto-batching, buffer management, error handling

### WLEDConnectionManager
- **Purpose**: Client lifecycle management
- **Manages**: Both python-wled and JSON clients
- **Features**: Pooling, cleanup, connection testing

### WLEDEffectBase
- **Purpose**: Base class for all effects
- **Provides**: Convenience methods for per-LED control
- **Features**: Automatic fallback, error tracking

### Effects (Meter, Rainbow Wave)
- **Purpose**: Actual effect implementations
- **Use**: Per-LED control for better visuals
- **Fallback**: Basic commands if JSON client unavailable
